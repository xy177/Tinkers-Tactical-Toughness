这个模组诞生的原因是我本想制作工匠防御的fork版本，但是因为工匠防御的体量过大并且存在部分未完成物品因此我将fork计划取消掉转而决定制作一个全新的mod来继承原来的想法计划，它的初始内容仍旧是与防御方面相关的但是会选用一些不同的思路——希望一切顺利。
目前计划：
1. 增加两种盾牌，拥有与Battlesign不同的定位与功能；
2. 或许会增加新的盔甲种类（如果我个人能力允许的话）；
3. 目前暂不打算增加新的材料，是否增加新的trait视情况而定；


This mod came about because I originally wanted to create a fork of Tinkers' Defense. However, due to its large scale and some unfinished content, I cancelled the fork plan and instead decided to make a completely new mod that carries on the original idea. Its initial content will still be defense-related, but with some different approaches — hopefully everything goes smoothly.

Current plans:
1. Add two types of shields with different roles and functions from Battlesign;
2. May add new types of armor (if my personal ability allows);
3. No plans to add new materials for now; whether to add new traits will be decided case by case.
