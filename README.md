I obtained a fork permission for this mod roughly a few years ago — and now I’ve decided to get started, though I have no idea how it will turn out (since some content beyond my ability may require assistance from Claude).
Here’s what I plan to do:
1. remove all original materials (as there are already plenty of add-ons for Tinkers' Construct that add new materials, such as Moartcon, Plustic, and others).

2. remove some poorly defined and seemingly unfinished tools (such as the saw and backpack), as well as some content that has already been implemented in other mods (such as rings and scissors).

3. rework the shield mechanics (I may reference the three shield designs from Tinkers' Construct 3; specifics will depend on the actual situation).

4. potentially rework certain weapons and tools, such as the Zwei and Mallet (this has very low priority)..

If anyone is reading this, it means I’ve progressed far enough to share the project publicly. If not… then let this project fade into obscurity.
